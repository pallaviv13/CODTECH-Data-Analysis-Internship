# Task 3 – Interactive Dashboard Development

##  Description

This task involves creating an interactive dashboard to visualize mobile sales data and generate actionable insights. The dashboard was developed using Power BI to present key performance indicators, sales trends, and customer behavior through dynamic and user-friendly visualizations.

---

##  Objective

* Design an interactive data visualization dashboard.
* Present business insights using graphical representations.
* Enable user-driven filtering and exploration of data.
* Demonstrate data storytelling using visualization tools.

---

##  Tools & Technologies

* Power BI Desktop
* Data Visualization Techniques
* CSV Dataset
* Business Intelligence Concepts

---

##  Dataset

**File Used:** `synthetic_mobile_sales_2025.csv`

The dataset contains mobile sales information including:

* Brand
* Country
* Units Sold
* Revenue (USD)
* Customer Rating
* Payment Method
* Monthly Sales Data

---

##  Dashboard Features

The dashboard includes the following visualizations:

* **Revenue by Brand** (Bar Chart)
* **Units Sold by Country** (Bar Chart)
* **Average Customer Rating** (KPI Card)
* **Monthly Revenue Trend** (Line Chart)
* **Payment Method Distribution** (Pie Chart)
* **Interactive Filters (Slicers)** for Brand and Country

---

##  Key Insights

* Certain mobile brands contribute significantly to overall revenue.
* Sales distribution varies across countries, indicating regional demand patterns.
* Customer ratings remain consistently positive.
* Digital payment methods are widely preferred.
* Monthly sales trends highlight changes in customer purchasing behavior.

---

##  Deliverable

A fully functional Power BI dashboard showcasing interactive visualizations and business insights derived from the dataset.
