# Task 1 – Big Data Analysis using PySpark

##  Description

This task demonstrates scalable data analysis using PySpark on a mobile sales dataset. The objective is to process structured data efficiently, perform exploratory analysis, and derive meaningful business insights using distributed computing concepts.

---

##  Objective

* Perform big data analysis using PySpark.
* Handle dataset processing efficiently.
* Generate analytical insights from sales data.
* Demonstrate scalability using Spark DataFrames.

---

##  Tools & Technologies

* Python
* PySpark
* Google Colab
* Pandas
* Matplotlib

---

##  Dataset

**File Used:** `synthetic_mobile_sales_2025.csv`

The dataset contains mobile sales information including:

* Brand
* Country
* Units Sold
* Revenue
* Customer Rating
* Payment Method
* Monthly Sales Data

---

##  Implementation Steps

1. Installed and configured PySpark in Google Colab.
2. Created Spark session for distributed processing.
3. Loaded dataset using Spark DataFrame.
4. Performed data exploration and validation.
5. Conducted aggregations and analytical queries.
6. Visualized key insights using charts.

---

##  Analysis Performed

* Revenue generated by each mobile brand.
* Units sold across different countries.
* Average customer ratings per brand.
* Monthly revenue trends.
* Payment method distribution.

---

##  Key Insights

* Certain brands contribute significantly to overall revenue.
* Sales vary across regions indicating market preferences.
* Customer ratings remain consistently positive.
* Digital payment methods are widely used.
* Monthly trends highlight fluctuations in demand.


##  Deliverable

A PySpark notebook showcasing scalable big data processing and analytical insights derived from the dataset.
