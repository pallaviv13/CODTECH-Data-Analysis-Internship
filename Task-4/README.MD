# Task 4 – Sentiment Analysis Using Natural Language Processing (NLP)

##  Description

This task focuses on performing sentiment analysis on textual data using Natural Language Processing (NLP) techniques. The project analyzes customer reviews and classifies sentiments as Positive, Negative, or Neutral to understand user opinions and overall product perception.

---

##  Objective

* Apply NLP techniques to analyze textual data.
* Perform sentiment classification using polarity analysis.
* Preprocess text data for analysis.
* Generate insights from customer reviews.

---

##  Tools & Technologies

* Python
* Google Colab
* TextBlob (NLP Library)
* Pandas
* Matplotlib

---

##  Dataset

A sample review dataset containing customer feedback text was created and analyzed.
The dataset includes:

* Customer Reviews (text data)
* Sentiment labels generated using NLP techniques

---

##  Implementation Steps

1. Created a dataset containing product reviews.
2. Installed and imported NLP libraries.
3. Applied text preprocessing techniques.
4. Calculated sentiment polarity using TextBlob.
5. Classified reviews into Positive, Negative, and Neutral categories.
6. Visualized sentiment distribution using charts.

---

##  Sentiment Analysis Method

**Library Used:** TextBlob

TextBlob calculates sentiment polarity values:

* Polarity > 0 → Positive Sentiment
* Polarity < 0 → Negative Sentiment
* Polarity = 0 → Neutral Sentiment

* Distribution of sentiments visualized using a bar chart.
* Insights derived from customer opinions.

##  Key Insights

* Negative feedback highlights areas for improvement.
* Sentiment analysis helps understand customer satisfaction levels.
* NLP techniques enable automated opinion analysis.
