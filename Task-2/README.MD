# Task 2 – Predictive Analysis Using Machine Learning

##  Description

This task focuses on building a Machine Learning model to predict customer rating categories using a mobile sales dataset. The project demonstrates data preprocessing, feature selection, model training, and performance evaluation using supervised learning techniques.

---

##  Objective

* Develop a predictive Machine Learning model.
* Perform feature engineering and data preparation.
* Train and evaluate a classification algorithm.
* Interpret model performance using evaluation metrics.

---

##  Tools & Technologies

* Python
* Google Colab
* Pandas & NumPy
* Scikit-learn
* Matplotlib

---

##  Dataset

**File Used:** `synthetic_mobile_sales_2025.csv`

The dataset includes mobile sales information such as:

* Brand and Country
* Units Sold
* Revenue (USD)
* Customer Rating
* Payment Method
* Monthly Sales Details

---

##  Implementation Steps

1. Loaded dataset into Google Colab.
2. Performed exploratory data analysis.
3. Created a target variable based on customer ratings.
4. Selected relevant numerical features.
5. Split data into training and testing sets.
6. Trained a Logistic Regression classification model.
7. Evaluated model performance using accuracy and classification metrics.

---

##  Machine Learning Model

**Algorithm Used:** Logistic Regression

The model predicts whether a customer rating belongs to a **High Rating** or **Low Rating** category based on sales-related features.

---

##  Model Evaluation

Evaluation metrics used:

* Accuracy Score
* Confusion Matrix
* Classification Report (Precision, Recall, F1-score)

---

##  Key Insights

* Revenue and units sold influence customer satisfaction trends.
* The classification model successfully predicts rating categories.
* The dataset provides meaningful patterns suitable for predictive analysis.
